Basic commands

Shortcut Keys Description
ctrl + s To save the file
ctrl + c To copy text and files
ctrl + v To paste the content
ctrl + x To cut the content
ctrl + a Selects all
ctrl + z To undo the edits
ctrl + y To redo edits
ctrl + "+" To zoom in the editor view
ctrl + "-" to redo zoom
shift + ">" to select content to right
shift + "<" to select content to left
shift + "up" to select and movie up
shift + "down" to select and movie down
alt + up it moves line one level up
alt + down it moves line one level down
shift + alt + "down" for copying same line
ctrl + shift + alt + 'down' for writting the code in multiple lines
ctrl + shift + n fro new VScode window
ctrl + shift + p for command pallete
ctrl + shift + w for closing the editor
ctrl + "/" for comments
