Html cheatsheet

S.No Tag Name Description
Structure
1. < !DOCTYPE html > Defines the document type and version of HTML.
2. < html › Root element of an HTML document.
3. < head › Container for metadata and links to scripts and stylesheets.
4. < title > Sets the document's title in the browser tab.
5. < body > Container for all the visible content on a webpage.
Meta Data
6. < meta › Provides metadata (character set, viewport, author, etc.)
7. < link > Links to external resources (stylesheets).
8. < style > Embeds internal CSS.
Text Content
9. < hi > to < h6 > Headings tags, < h1 > being the largest.
10. ‹ p › Used to define paragraph.
11. < br > Line breaks
12. < hr > Horizontal rule (thematic break).
13. < blockquote > Quotation
14. < pre > Preformatted text, preserves spaces and line breaks.
15. < b > Bold text.
16. < i > Italic text.
17. < u > Underlined text.
18. < strong > Important text (bold by default).
19. ‹ em > Emphasized text (italic by default).
20. ‹ small › Smaller Text.
21. < mark > Highlighted text.
22. ‹ del › Strikethrough text.
23. ‹ ins > Inserted text (underlined).
24. < sub > Subscript text.
25. < sup > Superscript text.
26. < code > Inline code.
27. < kbd > Keyboard input.
28. ‹ var > Variable.
Links And Media
29. < a > Anchor (hyperlink).
30. < img > Image tag
31. < figure > Self-contained content (e.g., image).
32. < figcaption > Caption for ‹ figure›.
33. < audio > Audio content.
34. < source > Media source (for < audio > and ‹ video >)
35. ‹ track > Text tracks (subtitles) for ‹ video >.
Lists
36. < ul > Unordered list.
37. < ol › Ordered list.
38. < li > List item.
39. < dl › Description list.
40. ‹ dt > Term (name) in a description list.
41. ‹ dd › Description in a description list.
Tables
42. < table > Table.
43. < thead › Table header group.
44. < tbody > Table body group.
45. ‹ tfoot > Table footer group.
46. < tr > Table row.
47. ‹ th > Table header cell.
48. ‹ td › Table data cell.
49. < caption > Table caption.
50. < colgroup > Group of columns.
51. < col › Column.
Forms
52. ‹ form > Form container.
53. < input > Input control.
54. < textarea > Multi-line text input.
55. < button > Clickable button.
56. < select > Drop-down list.
57. ‹ option > Option in a ‹ select› element.
58. < label › Label for input.
59. < fieldset › Group of related inputs.
60. < legend > Caption for ‹ fieldset›.
61. < datalist > List of pre-defined options for input.
Embeded Content
62. < iframe > Inline frame.
63. < embed > External content.
64. < object > Container for external resources.
65. < param > Parameters for < object ›
Scripting
66. < script > Client-side script.
67. < noscript > Alternative content when scripts are disabled.
Sectioning
68. < header > Introductory content.
69. < footer › Footer content.
70. < main > Main content.
71. < section› Thematic grouping of content.
72. < article > Independent, self-contained content.
73. < nav > Navigation links.
74. < aside > Side content (sidebar).
75. < div > Generic container for flow content.
Interactive Elements
76. < details> Disclosure widget.
77. < summary> Summary for ‹ details›.
78. < dialog> Dialog box.
79. < menu > Menu for commands or options.
Semantic Elements
80. < address> Contact information.
81. < time> Time and date.
