html emmet abbreviations

Emmet abbreviation Discription
child: > we can specify children element using '>'
sibling: + we can specify sibling element using '+'
climb up: ^ we can climb up the tree by using '^'
groupin: () we can crate header and footer without climbing using '()'
multiplication: * we can generate mulple tags by '*'
dollar sign: $ we can crate number of items in sequence using '$'
parenthesis: {} used to insert text
id and class atributes
#header creates a div with id="header"
.title create a class with name="title"
from#search.wide creates a form with id='search' classs='wide'
custom attributes
p[title="hello world"] creates a para with title hello world
td[rowspan=2 colspan=3 title] returns td with colspan and rowspan
implict tag names
.class creates a div with name class
ul.class creates ul with li class
table>.row>.col crates a table with classes row and col
